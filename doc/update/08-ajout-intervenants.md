# 08. Verification complete du module "Ajout des intervenants dans un projet en cours"

## Audit complet requis
Les elements suivants doivent etre controles :
- Creation des invitations
- Reception des invitations
- Acceptation
- Refus
- Attribution correcte des roles
- Gestion des permissions
- Notifications
- Acces aux differents modules apres validation

Cette fonctionnalite doit etre entierement fiable.
