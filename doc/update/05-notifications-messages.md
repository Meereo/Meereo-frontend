# 05. Notifications de nouveaux messages

## Probleme constate
Les utilisateurs ne sont pas correctement avertis lorsqu'ils recoivent un nouveau message. Cette anomalie concerne aussi bien les clients que les professionnels.

## Fonctionnement attendu
Les notifications doivent apparaitre :
- Dans la barre laterale
- Dans la section Messagerie
- Dans le centre de notifications
- Directement sur la conversation concernee

Le compteur des messages non lus doit etre mis a jour instantanement.
