# 06. Refonte du suivi des paiements

## Fonctionnement attendu
Le module financier doit offrir une transparence totale au client.

Chaque paiement effectue dans le cadre d'un projet doit automatiquement generer une notification.

Le client doit pouvoir consulter un historique detaille comprenant :
- Le montant
- La date
- L'heure
- Le beneficiaire
- Le moyen de paiement
- Le numero de transaction
- Le statut (en attente, valide, refuse, rembourse)
- Les justificatifs eventuels

L'objectif est que le client puisse suivre en temps reel toutes les operations financieres liees a son projet.
