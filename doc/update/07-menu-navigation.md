# 07. Amelioration du menu de navigation

## Probleme constate
Le menu lateral manque de visibilite. La section active est uniquement affichee avec un fond gris. Cette difference visuelle est insuffisante.

## Amelioration souhaitee
Creer un veritable etat actif comprenant :
- Une couleur de fond plus marquee
- Une barre verticale coloree
- Une icone mise en evidence
- Un texte plus contraste
- Eventuellement une legere animation

L'utilisateur doit identifier instantanement la section dans laquelle il se trouve.
